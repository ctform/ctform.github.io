leetcode两正序数组求中位数

发表于 2019-09-28 | 分类于 leetcode


这个题目乍一看似乎很简单，合并数组，使用Arrays.sort重新排序后根据数组长度直接输出中位数即可

时间复杂度为O(m+n)，空间复杂度为O(m+n)，而题目要求的是时间复杂度为log(m+n),其实看到log就联想到这个题目得使用二分法来解答。
